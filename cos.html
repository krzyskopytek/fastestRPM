Fastest RPM

« PreviousNext »
 Time
Long
 

Learning Objectives

  • Understand and play with gesture recognizers

Introduction

Goal: Build a single page application that visually displays the crazy speed at which your finger spins!

Setup

  • Create a new iOS project named: FastestRPM
  • Use the Single View template
  • Remember to push it to a new public repo on github

Some Extra Code to help you throughout the assignment's user stories

You might want to define some macros like these
   #define RADIANS(degrees) ((degrees * M_PI) / 180.0)
   #define MIN_DEGREES      -135.0
   #define MAX_DEGREES      135.0
   #define RANGE_DEGREES    (MAX_DEGREES - MIN_DEGREES)

   #define LIMIT_VELOCITY           7500.0 // Points per second
   #define LIMIT_VELOCITY_DELTA 10.0 // Points per second

   #define RESET_DELAY      0.1 // Seconds
   #define VELOCITY_DELAY   0.1 // Seconds
Get velocity from dimension components
        CGPoint componentVelocity = [sender velocityInView:self.view];
        CGFloat velocity = sqrt(pow(componentVelocity.x, 2) + pow(componentVelocity.y, 2));
Helper method for actually getting the needle to move. It's your job to use this method in the right place.
        - (void)moveNeedleWithVelocity:(CGFloat)velocity {

            self.currVelocity = velocity;

            // Calculate velocity of pan motion
            self.maxVelocity = MAX(self.maxVelocity, velocity);

            // Calculate proportion of current velocity to velocity limit
            CGFloat velocityProportion = velocity / LIMIT_VELOCITY;

            // Calculate proportion of RPM needle degree range
            CGFloat degrees = MIN(RANGE_DEGREES * velocityProportion, RANGE_DEGREES);

            // Move needle in degree range proportionate to velocity
            self.needleView.transform = CGAffineTransformRotate(self.minRotationTransform, RADIANS(degrees));
        }

MVP User Stories

As a user, I want to see a speedometer wheel

  • Create a new project named: FastestRPM
  • Use the Single View template
  • Create a new repository on GitHub (FastestRPM) and push the project to GitHub
  • Add a UIImageView to the view
  • Set the UIImageView's image to a speedometer image

As a user, I want to be able to see the speedometer needle

  • Add a UIImageView on top of the speedometer
  • Set the UIImageView's image to a needle image
  • As a user, I should initially see the needle at its base position (0 RPM).
  • Add a weak UIImageView property to the ViewController class
  • Connect the property to the needle
  • During setup (hint: viewDidLoad), change the properties of the needle image so that it angles to the 0 RPM position

As a user, I want to see the needle rise up when I make circular motions with one of my fingers.

  • Add a UIPanGestureRecognizer on top of the base view (highest parent view)
  • Make sure it accepts only one finger
  • Connect the outlet action to an appropriately named method
  • Calculate changes in the location of the touch points
  • For simplicity's sake, let's just calculate velocity from one point to the next (distance/time). HINT: http://en.wikipedia.org/wiki/Pythagorean_theorem
  • To change the image's angle, use transforms

As a user, I want to see the needle fall back naturally to its base position when I lift my fingers off

  • After the gesture state changes to ended, failed or cancelled, move the needle back to base position
  • Also, when the gesture hasn't been changed in 0.1 seconds, it should reset the needle's position (HINT: timers)

Stretch Goals

  • Make the dial display the highest velocity acheived in a single gesture.
